#define LOG_MASK    (LOG_INFO | LOG_WARN | LOG_ERROR)

LOG_MASK control whether to output log message.

Function log() will print the message only if (type & LOG_MASK == true).

0: disable 16: all 12: only WARN and ERROR 3: only INFO and SUCCESS
